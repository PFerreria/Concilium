# GRAPHVIZ INSTALLATION (Required for best diagram quality):
# 
# Windows:
#   Download from: https://graphviz.org/download/
#   OR: choco install graphviz
#   Add to PATH: C:\Program Files\Graphviz\bin
#
# macOS:
#   brew install graphviz
#
# Linux (Ubuntu/Debian):
#   sudo apt install graphviz
#
# Linux (RHEL/CentOS):
#   sudo yum install graphviz
#
# If Graphviz is not installed, the system will automatically
# fallback to matplotlib for diagram generation.
